# ENVAI Backend

Bu klasör, ENVAI platformunun ileride geliştirilecek backend servisleri için ayrılmıştır.

Planlanan mimari:

- FastAPI servisleri
- MySQL veritabanı
- REST API katmanı
- Kimlik doğrulama ve yetkilendirme
- ESG, karbon, CBAM, CSRD ve raporlama servisleri

Şimdilik bilinçli olarak boş bırakılmıştır. Bu aşamada backend kodu bulunmamaktadır.
