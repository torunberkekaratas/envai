# ENVAI Platform Frontend

React + Vite tabanlı kurumsal sürdürülebilirlik, ESG, karbon yönetimi ve uyumluluk SaaS ERP frontend uygulaması.

## Teknolojiler

- React 19+
- Vite
- React Router
- TailwindCSS
- Recharts
- React Icons
- Axios
- Zustand
- Framer Motion

## Windows

```bat
cd envai-platform\app
start-frontend.bat
```

## macOS / Linux

```bash
cd envai-platform/app
chmod +x start-frontend.sh
./start-frontend.sh
```

## Manuel Çalıştırma

```bash
npm install
npm run dev
```

Backend bağlantısı yapılmamıştır. Tüm ekranlar mock data ile çalışır.
